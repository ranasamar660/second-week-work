JavaScript Opretors

JavaScript has a variety of operators, which can be categorized into several types.Here are the main types of operators in JavaScript, along with examples.

JavaScript Opretors :

  1. Arithmetic Opretors
  2. Assignment operators
  3. Comparrion Opretors
  4. Logical operators
  1. Arithmetic operators
    • + (Addition)
    • - (Subtraction)
    • * (Multiplication)
    • / (Division)
    • % (Modulus)

1.   Addition + :

        let x = 5 + 3; //Addition  x = 8
        let x = 15 + 13; //Addition  x = 2
        let x = 25 + 33; //Addition  x = 58

2.   Subtraction - :

        let x = 5 - 3; //Subtraction x = 2 
        let x = 15 - 13; //Subtraction  x = 2
        let x = 25 - 33; //Subtraction  x = 2

3.   Multiplication * :

        let x = 5 * 3; //Multiplication x = 15 
        let x = 15 * 13; //Multiplication x = 195 
        let x = 25 * 33; //Multiplication  x = 825

4.   Division / :

        let x = 5 / 3; //Division x = 1.666666 
        let x = 15 / 13; //Division  x = 1.1538461538461537
        let x = 25 / 33; //Division x = 0.757575

5.   Modulus % :

        let x = 5 % 3; //remainder x = 2 
        let x = 15 % 13; //remainder  x = 2 
        let x = 3 % 9; //remainder x = 3
    2. Assignment operators
    • = (Assignment)
    • += (Addition Assignment)
    • -= (Subtraction Assignment)
    • *= (Multiplication Assignment)
    • /= (Division Assignment)
    • %= (Modulus Assignment)

1.   Assignment = :

       let x = 5; //  x = 5
       let a = samar; // a = samar

2.   Addition Assignment += :

       let x = 5; 
       x += 10; //  x = 15
       let a = 20;
       a += 10; // a = 30

3.   Subtraction Assignment -= :

       let x = 5; 
       x -= 2; //  x = 3
       let a = 20;
       a -= 10; // a = 10 
       let y = 10;
       y -= 11; //  y = -1

4.   Multiplication Assignment *= :

       let x = 5; 
       x *= 2; //  x = 10
       let a = 2;
       a *= 10; // a = 20 
       let y = 10;
       y *= 5; //  y = 50

5.   Division Assignment /= :

       let x = 10; 
       x /= 2; //  x = 5
       let a = 20;
       a /= 10; // a = 2 
       let y = 100;
       y /= 5; //  y = 20

6.   Modulus Assignment %= :

       let x = 11; 
       x %= 2; //  x = 1
       let a = 20;
       a %= 10; // a = 0 
       let y = 100;
       y %= 5; //  y = 0
    3. Comparison operators
    • == (Equal to)
    • === (strict Equal to)
    • != (Not Equal to)
    • !== (strict Not Equal to)
    • > (Greater Than)
    • < (Less Than)
    • >= (Greater Than or Equal to)
    • <= ( Less Than or Equal to)

1.   Equal to == :

        let isEqual = 5 == '5'; // isEqual is true
        let isEqual = 9 == '9'; // isEqual is true

2.   (strict Equal to) === :

        let isEqualStrict = 5 === '5'; // isEqualStrict is false 
        let isEqualStrict = 9 === '9'; // isEqualStrict is false

3.   (Not Equal to) != :

        let isNotEqual = 5 != '5'; // isNotEqual is false
        let isNotEqual = 9 != '9'; // isNotEqual is false

4.   (strict Not Equal to) !== :

        let isnotEqualStrict = 5 !== '5'; // isnotEqualStrict is true 
        let isnotEqualStrict = 9 !== '9'; // isnotEqualStrict is true

5.   (Greater Than) > :

        let isGreater = 10 > 3; // isGreater is true
        let isGreater = 1 > 3; // isGreater is false
        let isGreater = 11 > 3; // isGreater is true

6.   (Less Than) < :

        let isLess = 10 < 3; // isLess is false
        let isLess = 1 < 3; // isLess is true
        let isLess = 11 < 3; // isLess is false

7.   (Greater Than or Equal to) >= :

        let isGreaterThanOrEqualTo = 10 >= 5; // isGreaterThanOrEqualTo is true
        let isGreaterThanOrEqualTo = 2 >= 5; // isGreaterThanOrEqualTo is false

8.   ( Less Than or Equal to) <= :

        let isLesserThanOrEqualTo = 2 <= 5; // isLesserThanOrEqualTo is true
        let isLesserThanOrEqualTo = 10 <= 5; // isLesserThanOrEqualTo is false
    3. Logical operators
    • && (AND)
    • || (OR)
    • ! (NOT)

1.   AND &&:

        if ( true && true ) // true
        let andResult = (5 > 3) && (10 > 5); // andResult is true 
        if ( true && false ) // false
        let andResultFalse = (5 > 3) && (5 > 10); // andResultFalse is false
        if ( false && true ) // false
        let andResultFalse = (3 > 5) && (10 > 5); // andResultFalse is false

2.   OR || :

        if ( true || true ) // true
        let orResult = (5 > 3) || (10 > 5); // orResult is true
        if ( true || false ) // true
        let orResult = (5 > 3) || (5 > 10); // orResult is true 
        if ( false || false ) // false
        let orResultFalse = (5 < 3) || (15 < 10); // orResultFalse is false

3.   NOT !:

        if ( !false  ) // true
        let andResult = (5 != 3); // andResult is true 
        if ( !true ) // false
        let andResultFalse = (5 != 5); // andResultFalse is false
